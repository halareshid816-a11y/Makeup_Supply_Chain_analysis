# 💄 Makeup Supply Chain Analysis Project

## 📝 Overview
An end-to-end data analysis project focusing on the supply chain of a Beauty startup. The goal was to optimize logistics, analyze supplier performance, and maximize net profit.

## 🛠️ Tools Used
*   **Python (Pandas, Plotly):** For Data Cleaning, EDA, and Feature Engineering (Net Profit calculation).
*   **SQL (MySQL):** For data structuring and creating complex Views for reporting.
*   **Power BI:** For building an interactive dashboard to visualize KPIs and insights.

## 🚀 Key Insights
*   Identified **Skincare** as the most profitable category.
*   Detected high **Defect Rates** from specific suppliers (Supplier 1 & 5).
*   Analyzed **Shipping Costs vs. Time** to optimize carrier selection.

## 📂 Project Structure
*   `Untitled36.ipynb`: Python Notebook for data processing.
*   `final_summary_sql.csv`: SQL logic and data summary.
*   `SUPPLY CHAIN DASHBOARD.pbix`: The final Power BI report.
